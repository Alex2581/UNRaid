Hide Deprecated Applications=Приховати застарілі програми
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Цей параметр приховує застарілі програми зі списку доступних програм. Однак застарілі програми все одно відображатимуться в розділах встановлених та попередніх програм.
Hide Incompatible Applications=Приховати несумісні програми
This setting will hide applications that are listed as being incompatible with your version of Unraid=Цей параметр приховає програми, які вказані як несумісні з вашою версією Unraid
Allow install of second instance=Дозволити встановлення другого екземпляра
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Ця опція вмикає іншу кнопку на картках і спливаючих вікнах, яка дозволить вам переінсталювати встановлену програму, використовуючи її значення шаблону за замовчуванням. Цей параметр іноді може ввести в оману, і зазвичай використовується лише в тому випадку, якщо ви хочете встановити і запустити програму двічі одночасно.
Automatically open the sidebar=Автоматично відкривати бічну панель
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=Дозволяє CA автоматично відкривати бічну панель для отримання додаткової інформації при наведенні курсора на піктограму програми
Allow CA to check for updates to applications=Дозволити CA перевіряти наявність оновлень програм
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Цей параметр дозволяє CA перевіряти наявність оновлень для програм. Зауважте, що програми спільноти завжди перевірятимуть наявність оновлень для себе
Disable Mouse Wheel Page Changes=Вимкнути зміну сторінки коліщатком миші
This setting disables using left or right on a mouse wheel to change the page=Цей параметр вимикає використання лівого або правого колеса миші для зміни сторінки
Allow CA to send any emergency notifications=Дозволити CA надсилати будь-які екстрені сповіщення
This setting allows CA to periodically scan your system and alert you to the presence of=Цей параметр дозволяє CA періодично сканувати вашу систему та повідомляти про наявність
1 Known malware and security vulnerabilities within applications=1 Відомі зловмисне програмне забезпечення та вразливості безпеки в програмах
2 Critical security vulnerabilities within Unraid OS=2 Критичні вразливості безпеки в Unraid OS
3 Any installed applications which may seriously hinder the operation of your server=3 Будь-які встановлені програми, які можуть серйозно перешкоджати роботі вашого сервера
Save CA debugging information=Збережіть інформацію про налагодження CA
Only enable this when advised to from the author of Community Applications=Увімкніть це лише за порадою від автора програм спільноти
Enable developer mode=Увімкнути режим розробника
This unlocks some extra features within CA generally only used by developers Not recommended to enable=Це розблокує деякі додаткові функції в CA, які зазвичай використовуються лише розробниками. Не рекомендується вмикати
